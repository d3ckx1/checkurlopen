# checkurlopen

介绍：用于批量判断目标URL是否能访问，并且自动进行截图保存。

使用范围：安全服务中批量检查URL资产，内网渗透中，对探测到的http批量资产，进行自动化可用性访问并截图保存，解放双手。

后期还会继续追加新功能！！！！

使用方法：python checkurlopen.py ip.txt
